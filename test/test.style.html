html 스타일(style)


html 요소의 style 속성(attribut)을 이용하면 css 스타일을 html 요소에 직접 설정할 수 있습니다.
하지만 이러한 style 속성을 이용한 방법은 오직 단 하나의 html 요소에만 스타일을 적용할 수 있습니다.

문법

<태그 이름 style="속성 이름:속성값">
