display 속성값 : 블록

P요소

p요소는 display 속성값이 블록인 요소입니다.

div요소

div요소를 이용한 스타일 적용

div요소는 다른 html 요소들을 하나로 묵는데 자주 사용되는 대표적인 블록(block)요소 입니다.
div요소는 주로 여러 요소들의 스타일을 한 번에 적용하기 위해 사용됩니다.

인라인(inline)타입의 요소

display 속성값이 인라인(inline)인 요소는 새로운 라인(line)에서 시작하지 않습니다.
또한, 요소의 너비도 해당 라인 전체가 아닌 해당 html 요소의 내용(content)만큼만 차지합니다.
spad태그는 display 속성값이 인라인인 요소입니다.

span요소


span요소는 텍스트text)의 특정 부분을 묵는데 자주 사용되는 인라인(inline)요소 입니다.
span요소는 주로 텍스트의 특정 부분에 따로 스타일을 적용하기 위해 사용됩니다.

span요소를 이용한 스타일

이렇게 span요소로 텍스트의 일부분만을 따로 묶은 후에 스타일을 적용할 수 있습니다.
